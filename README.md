AI Mock Interview Coach

An AI-powered application that acts as a Hiring Manager to help students practice for job interviews.
Features

Resume Analysis: Upload your CV (PDF) and the AI analyzes it.
Job Description Matching: Paste a job link/description to tailor the interview.
Mock Interview: The AI asks realistic questions based on YOUR resume.
CV Writer Mode: Helps rewrite your resume to match job descriptions.

Tech Stack
Python
Streamlit (Frontend)
Ollama / Phi-3 (LLM Engine)
pypdf (PDF Reader)

How to Run Locally
Install Ollama and download the model:
ollama run phi3

Install Python libraries
pip install -r requirements.txt

Run the app:
streamlit run app.py
